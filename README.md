# 🎲 Flutter Dice App

A simple and modern **Dice Rolling App built with Flutter**.  
The app generates a random dice face every time the user taps the dice, providing a smooth and interactive user experience.

---

## ✨ Features
- Random dice roll generation
- Clean dark-themed UI
- Smooth state updates
- Asset-based dice face rendering
- Beginner-friendly Flutter project
